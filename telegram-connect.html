Connect your Telegram account

Connected Telegram profile
Telegram personal account posting is not handled by the Bot API. This page is built for bot-based posting to a Telegram channel or group. Make sure your bot is already added as an admin in the target channel or group before connecting.

Connection status

Not connected yet
Enter your bot token and target details, then verify the connection.
Public channel/group
Use the target username field, like @mychannel.
Private channel/group
Use the chat ID field, usually starting with -100.
Bot must be admin
Without posting permission, verification may pass but message sending will fail.

Quick setup

1
Create or use your Telegram bot
Get the token from BotFather.
2
Add the bot to your channel or group
Make it admin with posting rights.
3
Paste the token and target details here
Then click Verify & Connect.
4
Test before publishing from View
Use the sample post button.
